Kartenprovider

Setup einer interaktiven Karte mit Leaflet mit verschiedenen Kartendarstellung die von unterschiedlichen Kartenprovidern geliefert werden.

Nur das Beispiel anzeigen
In diesem schon etwas komplexeren Beispiel benötigen wir ein paar mehr Libraries um Stamen einzubinden (http://maps.stamen.com/js/tile.stamen.js), ebenso auch bei Google (http://maps.google.com/maps/api/js?v=3.2&sensor=false + https://raw.github.com/gist/2197042/2b90c41b39b7d5b3a851d8f256de2ebd3fe1fb74/leaflet-google.js) hinzu kommen eine HTML-Datei (beispielsweise index.html), eine JavaScript-Datei (bsp. map.js) und die Leaflet Bibliothek die man sich herunterladen kann.
index.html 
<!DOCTYPE html>
<html>
	<head>
		<meta charset="utf-8">
		<title>Leaflet Basic Setup</title>
		<link rel="stylesheet" type="text/css" href="leaflet/dist/leaflet.css">
		<script type="text/javascript" src="leaflet/dist/leaflet.js"></script>

		<script type="text/javascript" src="http://maps.stamen.com/js/tile.stamen.js?v1.2.0"></script>
		<script type="text/javascript" src="http://maps.google.com/maps/api/js?v=3.2&sensor=false"></script>
		<script type="text/javascript" src="https://raw.github.com/gist/2197042/2b90c41b39b7d5b3a851d8f256de2ebd3fe1fb74/leaflet-google.js"></script>

		<style>
			html, body { margin:0; padding: 0; }
			#map { min-height: 400px; }
		</style>
	</head>
	<body>
		<div id="map"></div>
		<script type="text/javascript" src="map.js"></script>
	</body>
</html>
map.js 
var map = L.map('map', {
	center : [52.52, 13.4],
	zoom : 13
});

var clm = L.tileLayer('http://{s}.tile.cloudmade.com/[API-Key]/997/256/{z}/{x}/{y}.png', {attribution: 'Map data © OpenStreetMap contributors, CC-BY-SA, Imagery © CloudMade'});
var osm = new L.TileLayer('http://{s}.tile.osmosnimki.ru/kosmo/{z}/{x}/{y}.png');
var mpn = new L.TileLayer('http://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png');
var qst = new L.TileLayer('http://otile1.mqcdn.com/tiles/1.0.0/osm/{z}/{x}/{y}.png', {attribution:'Tiles Courtesy of MapQuest '});
var wms = new L.TileLayer.WMS('http://wms.latlon.org/', {layers:'irs', crs: L.CRS.EPSG4326});
var gmr = new L.Google("ROADMAP");
var gms = new L.Google("SATELLITE");
var gmh = new L.Google("HYBRID");
var sto = new L.StamenTileLayer("toner");
var ste = new L.StamenTileLayer("terrain");
var stw = new L.StamenTileLayer("watercolor");
var tml = new L.TileLayer('http://a.tiles.mapbox.com/v3/[MapBox-API-Key]/{z}/{x}/{y}.png');

map.addLayer(osm);

map.addControl(new L.Control.Layers({'TileMill-via-MapBox':tml, 'CloudMade':clm, 'OSM':osm, 'Mapnik':mpn, 'MapQuest':qst, 'Scanex':wms, 'Google-Hybrid':gmh, 'Google-Satellite':gms, 'Google-Roadmap':gmr, 'Stamen-Toner':sto, 'Stamen-Terrain':ste, 'Stamen-Watercolor':stw}));
